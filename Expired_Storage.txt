

- 명칭

만료 시간이 추가된 데이터를 통한 저장 장치들의 기능 확장

- 배경

현재 저장 장치들의 데이터 삭제 방법

1. 임의의 데이터 또는 상반되는 데이터를 여러 차례 반복 저장
2. 데이터 삭제 시점에 해당 데이터 삭제 Command를 보내야 함
  2.1 Flash 메모리 장치인 경우, Trim, Secure Erase, Sanitize, Crypto Erase 방법 제공
3. 데이터를 삭제하려면 별도의 추가적인 정보 저장이 필요함

- 목적

데이터 기록 시점에서 만료 시간을 지정하여 프로세싱이 가능한 Flash 메모리 장치에 의해 자동적으로 삭제될 수 있도록 한다.

- 디바이스와 Flash 메모리 장치간의 시간 관리

1. 디바이스 시간에 절대적 의존, 초기화시에 암호화되어 전송된 시간 정보가 관리되어 만료 체크
2. Flash 메모리 장치의 초기화 과정에서 한번만 시간이 전송되며, 이후 Flash 메모리 장치의 타이머 장치에 의존함

- Flash 메모리 장치의 만료 시간 저장 공간 및 제거 방법

1. Flash 메모리 장치
 1.1 Expiring time을 Page의 Spare 공간을 활용하여 저장한다.
 1.2 Flash 메모리 장치에서 특정 Block들을 이용하여 저장한다.
 1.3 실시간 제거 or Background 제거
2. 디바이스 장치(Linux, Window)에서의 시간 전송 시스템 개발
